📋 SETUP (One Time - 5 minutes)
═══════════════════════════════════════════════════════════════

1. Get Zenodo Token
   https://zenodo.org/account/settings/applications
   → Create token with "deposit:write" scope

2. Set Token
   export ZENODO_API_TOKEN="your_token_here"

3. Test Setup
   ./scripts/test_zenodo_automation.sh
